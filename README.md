# 项目概述 python20

本项目是一个基于Python语言的财务管理系统，结合了Flask框架和MySQL数据库，旨在帮助用户高效管理个人财务，实现收支记录的数字化和智能化。

## 技术栈

- 开发语言：Python
- Web框架：Flask
- 数据库：MySQL
- 开发工具：PyCharm
- 数据库连接工具：Navicat

## 功能模块

- 用户注册与登录
- 收支记录的添加与管理
- 财务数据的统计分析
- 数据可视化展示
- 月度预算设置与实时监控

## 系统角色

- 普通用户：可注册、登录、管理个人财务数据

## 运行环境

- 操作系统：Windows
- 数据库系统：MySQL

## 部署步骤

1. 使用PyCharm导入项目
2. 安装项目依赖包
3. 配置MySQL数据库
4. 运行项目

## 目录结构

```
项目名/
│
├── app/ # Flask应用目录
│ ├── static/ # 静态资源目录
│ └── templates/ # HTML模板目录
│
├── venv/ # 虚拟环境目录
│
└── migrations/ # 数据库迁移文件目录
```

- 界面友好，操作简便
- 自动生成收支趋势和分类分布统计图表
